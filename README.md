# CRUD приложение для работы со списком пользователей

### Приложение представляет собой таблицу пользователей.

Стек: Vue 3, Pinia, TailwindCSS, PrimeVue;

#### Доступные действия над таблицей:
1) Добавить пользователя
2) Удалить пользователя
3) Редактировать данные пользователя
4) Отменить действия выполненные до сохранения

- Добавление происходит после нажатия на кнопку "Добавить", в таблице появляеться новая строка-черновик, с 3 инпутами на каждую строку "Имя", "Возраст", "Город". При нажатии на кнопку сохранить происходит валидация полей, если валидация успешна - запись попадает в таблицу, если нет, ошибки отображаются под инпутами и вылетает toast с ошибкой.

- Для Удаления нужно нажать на чекбокс слева от записи и нажать на кнопку удалить.

- Для Редактирования нужно нажать на чекбокс слева от записи, после чего появятся поля ввода, на месте записей, прямо в строке таблицы, с уже подставленными данными из таблицы. Для сохранения изменений нужно нажать кнопку "Сохранить", после чего пройдет валидация полей, если все успешно то данные отправяться в хранилище, если нет, то под полями ввода появятся ошибки и вылетит toast с ошибкой.

- Все изменения внесенные в поля ввода, а также выделение строки через чекбокс можно отменить по нажатию на кнопку "Отменить".

У таблицы есть пагинация, и переключатель в виде селектора, который позволяет отображать записи в кол-ве 5 (значение по умолчанию), 10, 20 штук на одной странице.
